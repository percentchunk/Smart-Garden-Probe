# Case Files

This folder contains the CAD files for the 3D-printed enclosure.

The case is a sandwich-style design with a top plate and bottom shell. It includes switch cutouts, an OLED window, a USB-C cutout for the XIAO RP2040, and space for the soil sensor cable.
